Shinkansen 隱私權政策

最後更新：2026 年 4 月 10 日

簡短版：Shinkansen 不會收集、儲存或傳送任何個人資料到我們的伺服器。您的所有設定與資料都只存在您自己的瀏覽器中。

1. 關於 Shinkansen

Shinkansen 是一款瀏覽器擴充功能，透過 Google Gemini API 將外語網頁翻譯成台灣繁體中文。本擴充功能需要使用者自行提供 Google Gemini API Key 才能運作。

2. 我們收集哪些資料

我們不收集任何資料。Shinkansen 沒有自己的伺服器，不會將您的任何資訊傳送給開發者。

3. 本地儲存的資料

以下資料儲存在您的瀏覽器本機，開發者無法存取：

API Key

您提供的 Google Gemini API Key 儲存在瀏覽器的 chrome.storage.local 中，僅用於向 Google Gemini API 發送翻譯請求。此金鑰不會傳送到 Google 以外的任何地方。

使用者偏好設定

翻譯相關設定（模型選擇、溫度參數、自動翻譯網站名單、術語表等）儲存在 storage.sync，僅透過瀏覽器的內建同步機制在您自己的裝置之間同步。

翻譯快取

已翻譯過的文字段落會快取在 chrome.storage.local，以避免重複呼叫 API。快取在擴充功能版本更新時自動清除。

用量統計

API 呼叫次數與 token 用量記錄在瀏覽器本機的 IndexedDB 中，用於在設定頁面顯示用量圖表。這些資料不會離開您的瀏覽器。

4. 傳送到第三方的資料

Shinkansen 唯一的對外網路連線是向 Google Gemini API（generativelanguage.googleapis.com）發送翻譯請求。每次翻譯時，以下資訊會傳送至 Google：

您的 Gemini API Key（作為身份驗證）、待翻譯的網頁文字段落，以及翻譯系統提示詞。

這些資料的處理受 Google 隱私權政策與 Gemini API 服務條款規範。建議您在使用前閱讀相關條款。

5. 我們不做的事

Shinkansen 不會追蹤您的瀏覽紀錄或瀏覽行為、不會收集個人識別資訊、不會將資料傳送到開發者或任何第三方伺服器（Google Gemini API 除外）、不會顯示廣告或使用分析追蹤工具、不會販售或分享您的任何資料。

6. 權限說明

本擴充功能要求以下瀏覽器權限：

<all_urls>（存取所有網站）

翻譯功能需要讀取並修改任意網頁的文字內容，因此需要此權限。擴充功能只在您主動觸發翻譯（或開啟自動翻譯的網域）時才會介入頁面。

storage（本地儲存）

用於儲存設定、API Key 與翻譯快取。

activeTab

用於在使用者主動啟動翻譯時，於當前作用中分頁啟用翻譯邏輯（內容腳本透過宣告式 content_scripts 注入，無需 scripting 動態權限）。

alarms

用於 service worker 排定閒置喚醒時機（例如維持翻譯串流連線、定期檢查更新），不背景收集資料。

7. 資料安全

您的 API Key 存放在瀏覽器的本地儲存區，受瀏覽器沙盒保護。我們建議您不要將 API Key 分享給他人，並在 Google AI Studio 中為您的 Key 設定適當的用量限制。

8. 兒童隱私

Shinkansen 並非針對 13 歲以下兒童設計。我們不會刻意收集兒童的個人資訊。

9. 政策變更

若本隱私權政策有重大變更，將隨擴充功能更新一併公告。您可以在本頁面查看最新版本。
