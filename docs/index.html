為什麼做 Shinkansen？

既有的沈浸式網頁翻譯工具會把你瀏覽的每一頁內容傳到第三方伺服器——無論你讀了什麼文章、看了什麼信件，全部都可能被看光

Shinkansen 的出發點就是保護您的隱私：

資料不外傳：翻譯請求直連 Google Gemini，不經過任何第三方伺服器
設定留在本地：所有資料只存在你自己的電腦上
原始碼完全公開：任何人都可以檢視程式碼，確認沒有後門

功能特色

保留網頁排版

原地替換文字，字型、字級、顏色、連結全部保留，看起來就像原生中文頁面

YouTube 字幕翻譯

看 YouTube 影片時，英文字幕自動翻譯為中文

跨頁自動延續翻譯

翻譯過的頁面點連結開新分頁、按返回鍵回上一頁都自動翻譯，不用每頁再按一次

雙翻譯引擎

Gemini + Google Translate 並存，重要內容用 Gemini，隨手瀏覽用 Google Translate 省費用
三組可自訂快速鍵（Alt+A / Alt+S / Alt+D），一鍵切換翻譯引擎

固定術語表

自訂翻譯對照表，確保專有名詞一律使用你指定的譯名

翻譯快取

翻過的段落自動快取，同一頁再開不花錢

費用透明

每次翻譯即時顯示花費多少錢，設定頁可回查過去每一筆翻譯紀錄的 token 用量與費用，支援圖表與 CSV 匯出

漸進式翻譯

分批翻譯、逐段注入，頁面逐步變成中文，進度一目瞭然

Beta 版手動安裝步驟

1

下載並解壓縮

從上方按鈕下載 .zip 檔案，解壓縮到任意位置

2

載入擴充功能

開啟 Chrome，前往 chrome://extensions/，打開右上角「開發人員模式」，點「載入未封裝項目」，選擇解壓縮後的 shinkansen 資料夾

3

設定 API Key

點工具列的 Shinkansen 圖示 → 設定 → 貼上你的 Gemini API Key完成！按 Alt+S 用 Gemini 翻譯，或按 Alt+D 用免費的 Google Translate（不需 API Key）
