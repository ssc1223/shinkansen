固定術語表

手動指定「原文 → 譯文」對照。每次翻譯時會注入 Gemini 的 system prompt，強制模型遵守你指定的譯法。固定術語的優先級高於自動擷取的術語

自動術語擷取

長文翻譯時，先呼叫 Gemini 擷取全文專有名詞建立對照表，確保分批翻譯的名詞譯名一致。短文（1 批以內）自動跳過。預設不開啟——開啟後術語翻譯會跳過 system prompt 的部分指示（例如「英文人名保留不翻」會失效）。詳細說明見 README

自訂 OpenAI 相容模型

想用 Gemini 之外的模型？設定一組 OpenAI 相容端點即可接 OpenRouter（含 Anthropic / Gemini / DeepSeek / Llama / Qwen / Grok 等百種模型）、 Ollama 本機、Together、Groq、Fireworks、OpenAI 自家等

設定完成後，到「一般設定」分頁的「翻譯快速鍵」，把任一組預設引擎改為「自訂模型」，即可使用快速鍵啟動翻譯

進階：自訂 request body 額外參數（JSON，覆蓋上方思考強度） 內容會深層 merge 進 chat.completions request body，可用來: (1) 強制覆蓋上方「思考強度」自動產生的設定； (2) 加 provider 專屬參數（例如 OpenRouter 的 top_k / min_p、 Anthropic 的 metadata)。 格式錯誤會在 Debug 分頁 log 一條 warn 並忽略，翻譯不會中斷

模型計價（USD）

請填入 input / output 單價， 系統會用來估算翻譯費用顯示在翻譯完成通知與「用量紀錄」分頁。填 0 代表不顯示費用 （token 數仍會記錄）

禁用詞清單

針對 LLM 容易漏網的中國大陸用語自訂禁用對照表。內容會以高顯著性區塊 注入 Gemini 的 system prompt 末端，明確要求譯文不可使用左欄詞彙、 必須改用右欄。修改清單後會自動讓既有翻譯快取分區，不再命中舊譯文
